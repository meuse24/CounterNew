This app allows you to count and log two different types of events. Each event is automatically timestamped and stored securely on your device.

Basic Usage:

  • Record events by tapping either of the two event buttons at the top
  • View your event history in chronological order below the buttons
  • Delete individual events by tapping the X icon on any event

Menu Options:

  • Share as CSV: Export your event history as a CSV file for analysis
  • Reset: Clear all recorded events (requires confirmation)
  • Event Types: Customize the labels for both event buttons
  • Info: View this help screen

Customization:

  • Access Event Types in the menu to change button labels
  • Each button can have its own custom label
  • Labels are saved automatically and persist between app restarts
Data Management
  • All events are saved automatically
  • Use the CSV export feature for backup or analysis
  • Data is stored locally on your device
