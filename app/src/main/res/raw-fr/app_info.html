Cette application vous permet de compter et d’enregistrer deux types d’événements différents. Chaque événement est automatiquement horodaté et stocké de manière sécurisée sur votre appareil.

Utilisation de base :

  • Enregistrez des événements en appuyant sur l’un des deux boutons d’événement situés en haut de l’écran
  • Consultez l’historique de vos événements dans l’ordre chronologique sous les boutons
  • Supprimez des événements individuels en appuyant sur l’icône « X » de l’événement concerné

Options du menu :

  • Partager en CSV : exportez l’historique de vos événements au format CSV pour analyse
  • Réinitialiser : supprime tous les événements enregistrés (confirmation requise)
  • Types d’événements : personnalisez les étiquettes des deux boutons d’événement
  • Info : affiche cet écran d’aide

Personnalisation :

  • Accédez à « Types d’événements » dans le menu pour changer les libellés des boutons
  • Chaque bouton peut avoir son propre libellé personnalisé
  • Les libellés sont enregistrés automatiquement et conservés lors des redémarrages de l’application
Gestion des données
  • Tous les événements sont enregistrés automatiquement
  • Utilisez la fonction d’export CSV pour la sauvegarde ou l’analyse
  • Les données sont stockées localement sur votre appareil
